1. Общий обзор системы

Система СОЮЗ предназначена для учёта топливных карт и контроля операций по заправке. Администратор имеет полный доступ ко всем функциям системы.

Основные возможности:
  • Создание и редактирование клиентов, операторов АЗС
  • Выпуск топливных карт с кодом, индексом, балансом и дневным лимитом
  • Блокировка и разблокировка карт с указанием причины
  • Пополнение баланса карт
  • Учёт операций (заправка, пополнение, списание, оприходование)
  • Пересчёт балансов по истории операций
  • Контроль дневных лимитов заправки
  • Управление справочниками (АЗС, виды топлива с единицами измерения)
  • Просмотр кабинетов клиентов из панели администратора
  • Панель оператора АЗС с поддержкой сканера штрихкодов
  • Интеграция с 1С через API

2. Управление клиентами

2.1. Просмотр и фильтрация

На вкладке "Клиенты" отображается список клиентов. Переключатель вверху позволяет выбрать режим: Клиенты или Администраторы. Есть строка поиска по ИНН или названию.

2.2. Добавление клиента

  1. Нажмите + Добавить
  2. Заполните поля: ИНН, Название, Адрес, Телефон, Email, Логин, Пароль
  3. При необходимости отметьте "Это администратор", а для операторов АЗС — дополнительно "Оператор АЗС"
  4. Нажмите Создать

2.3. Ярлык для оператора

Для учётных записей операторов доступна кнопка Ярлык для оператора (иконка ссылки). В диалоге выберите АЗС, введите пароль оператора — система сформирует ссылку, которую можно скопировать и открыть сразу в нужном режиме на планшете или ПК кассира.

2.4. Просмотр кабинета клиента

Кликните по названию клиента в таблице — откроется его кабинет от лица администратора. Кнопка "Админпанель" вернёт вас обратно.

3. Управление картами

3.1. Нумерация карт: код и индекс

Новое: система кода/индекса

Каждая карта идентифицируется парой код / индекс, например 0002/1. Это позволяет выпускать несколько карт с одинаковым кодом (например, для разных видов топлива одного клиента):

  • 0002/0 и 0002/1 — допустимо
  • 0002/1 и 0002/1 — недопустимо, система выдаст предупреждение

Индекс — число от 0 до 9. При стандартном использовании оставляйте индекс 0.

3.2. Просмотр и фильтрация

Вкладка "Карты" содержит три фильтра:

ФильтрОписание
Код картыТекстовый ввод — фильтрует по вхождению кода (например, введите «0002» чтобы увидеть все карты с этим кодом)
КлиентВыбор из списка клиентов
Вид топливаВыбор из списка видов топлива

Таблица карт отображает код/индекс, клиента, вид топлива, баланс, дневной лимит, статус и PIN-код. Единицы измерения (л, ₽, кг, шт) берутся из настроек вида топлива.

3.3. Создание карты

  1. Нажмите + Добавить
  2. Введите Код карты (4 цифры, например 0005)
  3. Выберите Индекс (0–9). В списке отображается итоговый вид карты: 0005/0
  4. Выберите Клиента и Вид топлива
  5. Укажите Начальный баланс (если > 0, автоматически создаётся операция пополнения)
  6. Введите PIN-код и при необходимости Дневной лимит
  7. Нажмите Создать
Защита от дублирования: если карта с таким кодом и индексом уже существует, появится окно с предупреждением. Нажмите "Изменить данные" чтобы вернуться к форме и выбрать другой индекс.

3.4. Пополнение баланса

  1. В строке нужной карты нажмите кнопку +
  2. Введите количество и цену — сумма рассчитается автоматически (или наоборот)
  3. При необходимости добавьте комментарий
  4. Нажмите Пополнить

3.5. Блокировка и разблокировка

Используйте кнопки 🔒 и 🔓 в строке карты. При блокировке обязательно укажите причину — она отобразится в кабинете клиента и в таблице карт.

3.6. Редактирование карты

Кнопка ✏️ открывает форму редактирования. Можно изменить код, индекс, PIN, лимит, статус. При изменении кода/индекса также срабатывает проверка на дублирование.

Важно: вид топлива можно изменить только если баланс карты равен нулю.

3.7. Пересчёт баланса

Кнопка ↺ пересчитывает баланс карты на основе всей истории операций. Используйте при расхождении баланса с историей.

4. Управление операциями

4.1. Типы операций

ТипВлияние на балансКогда использовать
Пополнение+Зачисление топлива на карту
Заправка−Отпуск топлива через АЗС
Списание−Ручное уменьшение баланса
Оприходование+Перемещение между картами (приход)

4.2. Фильтрация операций

Доступны фильтры: Карта, АЗС, Тип операции, Дата с, Дата по. Используйте их для формирования отчётов за период.

4.3. Добавление операции вручную

  1. Нажмите + Добавить
  2. Выберите карту, АЗС, дату, тип операции
  3. Укажите количество, цену и сумму
  4. Нажмите Создать
Совет: операции из панели оператора создаются автоматически. Ручное добавление используйте для корректировок и переносов.

5. Управление АЗС

Вкладка "АЗС" — справочник станций. Для каждой указываются наименование, Код 1С и адрес.

Важно: АЗС с названием, содержащим «Склад», используется как внутренняя точка для пополнений и перемещений. Она не отображается операторам в панели выбора АЗС.

6. Виды топлива и единицы измерения

Новое: единицы измерения

Каждый вид топлива теперь имеет поле Ед. изм.. Доступные значения:

  • л — литры (по умолчанию для топлива)
  • ₽ — рубли (для денежных карт)
  • кг — килограммы (для газа)
  • шт — штуки (для прочих позиций)

Выбранная единица автоматически отображается во всех балансах, лимитах и операциях — в панели администратора, кабинете клиента и панели оператора.

Добавление вида топлива

  1. Перейдите на вкладку "Виды топлива"
  2. Нажмите + Добавить
  3. Введите название, код 1С и выберите единицу измерения
  4. Нажмите Создать

7. Панель оператора

Панель оператора доступна по адресу /operator. Она предназначена для работы на АЗС.

7.1. Вход

Оператор выбирает АЗС из списка, вводит логин и пароль. Сессия сохраняется, при следующем открытии страницы вход выполняется автоматически.

Для удобства администратор может сформировать ссылку-ярлык (раздел «Клиенты» → кнопка ссылки у оператора), которая открывает панель уже авторизованной.

7.2. Отпуск топлива

  1. Поднесите штрихкод карты к сканеру или введите 4-значный код вручную
  2. Если найдена одна карта — сразу отображается информация по карте: код/индекс, клиент, вид топлива, баланс, лимит, доступный остаток
  3. Если найдено несколько карт с одинаковым кодом — появляется окно выбора в виде сетки 3×3. Каждый слот показывает код/индекс, вид топлива, баланс, лимит и доступный остаток. Нажмите на нужную карту
  4. Введите количество для отпуска — с клавиатуры или через экранный цифровой блок
  5. Нажмите "Отпустить топливо" и подтвердите действие
Единицы измерения в панели оператора соответствуют настройкам вида топлива карты. Для литрового топлива — «л», для рублёвых карт — «₽» и т.д.
Карта 0000 является системной и не может быть использована для отпуска топлива через панель оператора.

8. Полезные советы и интеграция с 1С

8.1. Интеграция с 1С

Система поддерживает синхронизацию с 1С через код 1С, который указывается для каждой АЗС и каждого вида топлива. Используйте эти коды при настройке обмена данными.

8.2. Печать и экспорт

На каждой вкладке доступна кнопка Печать. В кабинете клиента и истории операций доступен Экспорт в Excel.

8.3. Пересчёт баланса

Если баланс карты не совпадает с суммой операций, используйте кнопку ↺ Пересчитать баланс в строке карты. Система пересчитает баланс по всей истории и обновит значение.

8.4. Рекомендации по нумерации карт

СитуацияРекомендация
Одна карта одному клиентуИспользуйте индекс 0 (по умолчанию)
Несколько видов топлива по одному кодуИспользуйте разные индексы: 0, 1, 2...
Карта утеряна, нужна заменаЗаблокируйте старую, создайте новую с тем же кодом и следующим индексом

8.5. Системная карта 0000

Карта с кодом 0000 используется для учёта общего баланса склада. Она отмечается специальным значком и не предназначена для отпуска через панель оператора.
